Linux Command Line Basics

This guide provides a practical overview of fundamental Linux commands, file navigation, permissions, and system management. It is designed for IT support, help desk, and junior system administrator roles where command-line proficiency is essential.

Understanding the Linux Filesystem Hierarchy

Understanding how Linux organizes its files and directories is essential before learning command-line navigation.Linux follows a standardized filesystem layout defined by the Filesystem Hierarchy Standard (FHS). This structure ensures consistency across distributions and helps users know where system files, configuration files, and user data are stored.

Key Directories and Their Roles

Linux organizes files into a structured hierarchy, where each directory has a specific purpose. Understanding the key directories helps navigate, manage, and maintain the system efficiently.

/etc — System configuration
/home — User home directories
/var — Logs & dynamic system data
/usr — Applications, binaries, libraries
/bin — Essential user commands
/sbin — System administration commands
/lib — Shared libraries
/tmp — Temporary files
/opt — Optional/third-party software
/root — Root user’s home
/mnt — Mounted filesystems
/media — Removable media

Linux Command Line Basics

Working in the terminal allows you to interact directly with the system, performing tasks quickly and precisely. These are some of the many commands used within the Linux command line.

File & Directory Commands

ls → List files in a directory
cd → Change directory
pwd → Show current directory path

File Viewing & Editing

cat → Display file contents
nano → Edit a file in terminal
less → View file page by page
touch → Create a blank file

Process Management

ps → List running processes
top → Interactive process viewer
kill → Terminate a process
htop → Interactive process monitor (if installed)

Networking

ping → Test connectivity to a host
ifconfig → Show network interfaces
ssh → Connect to remote systems
curl → Transfer data from URLs

Summary

This guide covers essential Linux commands and workflows that IT support and help desk technicians use daily. Mastery of these fundamentals prepares you for troubleshooting, system navigation, and eventually more advanced system administration tasks.
